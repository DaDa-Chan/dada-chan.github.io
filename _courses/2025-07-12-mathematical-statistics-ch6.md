# 第六章 非参数假设检验

## §6.1 符号检验

设 $$X\sim F(x)$$ 为连续型随机变量，X的p分位数 $$t_{p}$$ 满足 $$F(t_{p})=p$$。

#### 假设检验问题
考察假设检验问题，例如：
$$H_{0}:t_{p}\le t_{0}\leftrightarrow H_{1}:t_{p}>t_{0}$$

#### 检验统计量与方法
设 $$\tilde{X}=(X_{1},\cdots,X_{n})$$ 是来自总体X的一个样本，令
$$Y_{i}=\begin{cases}1, & X_{i}-t_{0}>0\\ 0, & \text{其它}\end{cases}$$
并设 $$\theta=P(Y_{i}=1)=P(X_{i}-t_{0}>0)$$。则 $$Y_{1},\cdots,Y_{n}$$ i.i.d. $$\sim B(1,\theta)$$，且 $$\sum_{i=1}^{n}Y_{i}\sim B(n,\theta)$$。
$$\sum_{i=1}^{n}Y_{i}$$ 恰好是 $$\{X_{i}-t_{0}\}$$ 中正值的个数，记为 $$N^{+}$$。

原假设检验问题等价于关于$$\theta$$的检验问题。例如，$$H_{0}:t_{p}\le t_{0}$$ 等价于 $$H_{0}:\theta\le1-p$$。

#### 拒绝域

| $$H_{0}$$ | $$H_{1}$$ | 拒绝域 |
| :--- | :--- | :--- |
| $$t_{p}\le t_{0}$$ | $$t_{p}>t_{0}$$ | $$\{\tilde{x}:N^{+}\ge C^{*}\}$$, 其中 $$C^{*}=min\{C\in\mathcal{N}:\sum_{i=C}^{n}\binom{n}{i}(1-p)^{i}p^{n-i}\le\alpha\}$$ |
| $$t_{p}\ge t_{0}$$ | $$t_{p}<t_{0}$$ | $$\{\tilde{x}:N^{+}\le C^{*}\}$$, 其中 $$C^{*}=max\{C\in\mathcal{N}:\sum_{i=0}^{C}\binom{n}{i}(1-p)^{i}p^{n-i}\le\alpha\}$$ |
| $$t_{p}=t_{0}$$ | $$t_{p}\ne t_{0}$$ | $$\{\tilde{x}:N^{+}\le C_{1}^{*} \text{ 或 } N^{+}\ge C_{2}^{*}\}$$, 其中 $$C_1^*, C_2^*$$ 按显著性水平$$\alpha/2$$确定 |

## §6.2 秩和检验 (Wilcoxon Test)

### 一、基本概念

**定义 (秩)**: 设 $$x_{1}, \dots, x_{n}$$ 为两两互不相等的实数，若其中恰有个元素的值不超过 $$x_i$$ (包括自己)，则称 $$x_i$$ 在 $$(x_{1},\cdots,x_{n})$$ 中的**秩** (rank)为 $$R_i$$。

**定义 6.2.1**: 设 $$X_{1}, \dots, X_{n}$$ 为两两互不相等的一组样本，将其从小到大排列成 $$X_{(1)}<X_{(2)}<\cdots<X_{(n)}$$，若 $$X_{i}=X_{(R_{i})}$$，则称 $$X_i$$ 在样本中的**秩**为 $$R_i$$。

**定义 6.2.2**: 设 $$X_{1}, \dots, X_{n}$$ 为样本，$$R_{i}$$ 为 $$X_{i}$$ 的秩，则称 $$R=(R_{1},\cdots,R_{n})$$ 为 $$(X_{1},\cdots,X_{n})$$ 的**秩统计量** (rank statistics)。基于秩统计量的检验方法称为**秩检验** (rank test)。

**定理**: 设 $$X_{1}, \dots, X_{n}$$ 为来自连续型总体 $$X\sim F(x)$$ 的简单随机样本，则其秩统计量R取 $$(1,2,\cdots,n)$$ 的任一置换的概率都为 $$1/n!$$。

### 二、秩和检验
为比较两连续总体 $$X\sim F(x)$$ 和 $$Y\sim G(y)$$，从两个总体中分别抽取独立样本 $$X_{1},\dots,X_{m}$$ 和 $$Y_{1},\dots,Y_{n}$$。记$$Y_i$$在合样本中的秩为$$R_i$$。

**检验统计量**:
$$W=\sum_{i=1}^{n}R_{i}$$

**拒绝域的形式**:

| $$H_{0}$$ | $$H_{1}$$ | 拒绝域 |
| :--- | :--- | :--- |
| $$F(x)\le G(x)$$ | $$F(x)>G(x)$$ | $$W\ge c$$ |
| $$F(x)\ge G(x)$$ | $$F(x)<G(x)$$ | $$W\le d$$ |
| $$F(x)=G(x)$$ | $$F(x)\ne G(x)$$ | $$W\le d$$ 或 $$W\ge c$$ |

#### 临界值的确定

**秩和统计量的性质**:
* 记 $$W_X, W_Y$$ 分别为X样本和Y样本在合样本中的秩和，则 $$W_{X}+W_{Y}=(m+n)(m+n+1)/2$$ 
* 当 $$F(x)=G(x)$$ 时，W的分布关于 $$n(m+n+1)/2$$ 对称 
* $$\sup_{F(x)\le G(x)} P\{W\ge c \mid X_{i}\sim F,Y_{j}\sim G\}=P_{F(x)=G(x)}\{W\ge c\}$$
* $$\sup_{F(x)\ge G(x)} P\{W\le d \mid X_{i}\sim F,Y_{j}\sim G\}=P_{F(x)=G(x)}\{W\le d\}$$ 

**小样本情形**: 当m和n不太大时，可查表确定临界值 $$c_\alpha$$ 和 $$d_\alpha$$。
**大样本情形**:
**定理**: 假设 $$F(x)\equiv G(x)$$，则
* $$EW=\frac{n(m+n+1)}{2}$$
* $$Var\{W\}=\frac{mn(m+n+1)}{12}$$
* 当 $$m,n\rightarrow\infty$$ 时，
    $$W^{*}=\frac{W-n(m+n+1)/2}{\sqrt{mn(m+n+1)/12}}\rightarrow N(0,1)$$
    当m, n充分大时（一般大于7即可），取 $$W^*$$ 为检验统计量，并使用正态分布的分位数确定拒绝域。

| $$H_{0}$$        | $$H_{1}$$        | 拒绝域                     |
| :------------- | :------------- | :---------------------- |
| $$F(x)\le G(x)$$ | $$F(x)>G(x)$$    | $$W^{*}\ge u_{\alpha}$$   |
| $$F(x)\ge G(x)$$ | $$F(x)<G(x)$$    | $$W^{*}\le-u_{\alpha}$$   |
| $$F(x)=G(x)$$    | $$F(x)\ne G(x)$$ | $$W^{*}\ge u_{\alpha/2}$$ |

## §6.3 成对数据的检验问题——样本问题中的非参数假设检验

对于成对数据 $$(X_i, Y_i)$$，记 $$Z_i = X_i - Y_i$$。检验问题 $$H_{0}:F(x)=G(x)$$ 等价于检验 $$Z_i$$ 的分布是否关于原点对称。

#### 成对数据的符号检验法:
* **检验统计量**: $$N^{+}=\sum_{i=1}^{n} I\{Z_{i}>0\}$$，即 $$\{Z_{i}\}$$ 中正值的个数。
* **$$H_0$$下的分布**: 在$$H_0$$为真时，$$N^{+}\sim B(n,1/2)$$。
* **拒绝域**: 对于双边检验 $$H_1: F(x) \ne G(x)$$，拒绝域为 $$D=\{N^{+}\ge c \text{ 或 } N^{+}\le d\}$$。

#### 成对数据的符号秩和检验法:
* **检验统计量**:
    $$W^{+}=\sum_{i=1}^{n}V_{i}R_{i}$$
    其中 $$V_{i}=I\{Z_i>0\}$$，$$R_i$$ 为 $$\lvert Z_{i}\rvert$$ 在 $$(\lvert Z_{1}\rvert,\cdots,\lvert Z_{n}\rvert)$$ 中的秩。
* **拒绝域**: 对于双边检验，拒绝域为 $$\{W^{+}\le d \text{ 或 } W^{+}\ge c\}$$。

**$$W^+$$的性质**:
* **精确分布**: 在 $$H_0$$ 下，$$P(W^{+}=i)=\frac{t_{n}(i)}{2^{n}}$$，其中 $$t_n(i)$$ 为在 $$1,\dots,n$$ 中取若干数其和为i的方法数。
* **大样本情形**:
    * $$EW^{+}=\frac{n(n+1)}{4}$$
    * $$Var\{W^{+}\}=\frac{n(n+1)(2n+1)}{24}$$
    * 当 $$n\rightarrow\infty$$ 时，
        $$(W^{+})^{*}=\frac{W^{+}-n(n+1)/4}{\sqrt{n(n+1)(2n+1)/24}}\rightarrow N(0,1)$$


## §6.4 χ²拟合优度检验

### 一、分类数据的检验

#### 1. 总体理论分布不含未知参数
设总体X可以分成r类: $$A_{1},A_{2},\cdots,A_{r}$$，要检验的假设为
$$H_{0}:P(A_{i})=p_{i}, \quad i=1,2,\cdots,r$$
其中各个$$p_i$$已知, $$\forall p_{i}\ge0, \sum_{i=1}^{r}p_{i}=1$$。
对总体作了n次观察，各类出现的频数分别为 $$n_{1},n_{2},\cdots,n_{r}$$，且 $$\sum_{i=1}^{r}n_{i}=n$$。

**检验统计量**:
$$\chi^{2}=\sum_{i=1}^{r}\frac{(n_{i}-np_{i})^{2}}{np_{i}}$$
有时也用下式计算：
$$\chi^{2}=\sum_{i=1}^{r}\frac{n_{i}^{2}}{np_{i}}-n$$

**定理**: 在$$H_0$$为真时，
$$\chi^{2}\rightarrow\chi^{2}(r-1), \quad \text{as } n\rightarrow\infty$$
**拒绝域**: 对于给定的显著性水平$$\alpha$$，当n较大时，可取拒绝域为
$$D=\{\chi^{2}>\chi_{\alpha}^{2}(r-1)\}$$
这一检验一般在 $$n\ge50$$ 及 $$np_i \ge 5$$ ($$n_i \ge 5$$) 的条件下进行。

#### 2. 总体理论分布含未知参数
设总体X可以分成r类: $$A_{1},A_{2},\cdots,A_{r}$$，要检验的假设为
$$H_{0}:P(A_{i})=p_{i}, \quad i=1,2,\cdots,r$$
其中各个 $$p_{i}=p_{i}(\theta_{1},\cdots,\theta_{m})$$ 依赖于m个未知(自由)参数。

先求出 $$\theta_{1},\cdots,\theta_{m}$$ 的极大似然估计 $$\hat{\theta_{1}},\cdots,\hat{\theta_{m}}$$，从而得到$$p_i$$的估计 $$\hat{p_{i}}=p_{i}(\hat{\theta_{1}},\cdots,\hat{\theta_{m}})$$。
**检验统计量**:
$$\chi^{2}=\sum_{i=1}^{r}\frac{(n_{i}-n\hat{p}_{i})^{2}}{n\hat{p}_{i}}$$
**定理 (R. A. Fisher)**: 在$$H_0$$为真时，
$$\chi^{2}\rightarrow\chi^{2}(r-m-1)$$
$$\chi^2$$分布的自由度 = (分类数) - (未知参数空间的维数) - 1。
**拒绝域**:
$$D=\{\chi^{2}>\chi_{\alpha}^{2}(r-m-1)\}$$

### 二、分布拟合的检验
要检验的原假设为
$$H_{0}: \text{总体分布 } F(x)=F_{0}(x)$$
其中 $$F_0(x)$$ 称为理论分布，它可以是一个完全已知的分布，也可以是依赖于有限个未知参数且数学形式已知的分布。

#### 1. 离散型分布
若理论分布 $$F_0(x)$$ 是离散型的，将所有可能取值并成有限个类 $$B_{1},\cdots,B_{r}$$，并统计样本落入每个$$B_i$$的频数$$n_i$$。记
$$P(X\in B_{i} \mid H_{0})=p_{i}=p_{i}(\theta)$$
问题转化为分类数据的检验问题。

#### 2. 连续型分布
检验 $$H_0: X$$ 服从分布 $$F_0(x)$$的做法如下：
* 将 $$F_0(x)$$ 的取值范围分成r个区间 $$A_{1},\dots,A_{r}$$。
* 统计样本落入这r个区间的频数 $$n_{1},\dots,n_{r}$$。要求 $$n_i \ge 5$$。
* 记 $$p_{i}=P(X\in A_{i} \mid H_{0})=F_{0}(a_{i})-F_{0}(a_{i-1})$$。
* 若有未知参数，用其极大似然估计值代入计算$$\hat{p}_i$$。
* 这样问题就转化为分类数据的检验问题，并按相应步骤进行检验。

## §6.5 列联表的独立性检验
假定n个随机试验结果(或n个个体)根据两个特性 $$X_1$$ 和 $$X_2$$ 进行分类，若 $$X_1$$ 中有r类 $$A_1, \dots, A_r$$ ($$r \ge 2$$)，$$X_2$$ 中有s类 $$B_1, \dots, B_s$$ ($$s \ge 2$$)，属于 $$A_i$$ 和 $$B_j$$ 的个体的数目为 $$n_{ij}$$，那么得到 r x s **列联表**。

记总体为X，它是二维变量 $$(X_1, X_2)$$。设
$$P(X\in A_i \cap B_j) := P(X_1 \in A_i, X_2 \in B_j) = p_{ij}, \quad i=1,\dots,r; j=1,\dots,s$$
又记边际概率为：
$$p_{i\cdot} = P(X_1 \in A_i) = \sum_{j=1}^{s} p_{ij}$$
$$p_{\cdot j} = P(X_2 \in B_j) = \sum_{i=1}^{r} p_{ij}$$

要检验的是两个特性 $$X_1$$ 和 $$X_2$$ 是否独立。其假设检验问题为：
$$H_0 : p_{ij} = p_{i\cdot}p_{\cdot j}, \quad \forall i,j$$
$$H_1 : \text{至少一对(i,j)使 } p_{ij} \ne p_{i\cdot}p_{\cdot j}$$

#### 1. 边际概率 $$p_{i\cdot}, p_{\cdot j}$$ 已知时
* **检验统计量**:
    $$\chi^{2}=\sum_{i=1}^{r}\sum_{j=1}^{s}\frac{(n_{ij}-np_{i\cdot}p_{\cdot j})^{2}}{np_{i\cdot}p_{\cdot j}}$$
* **$$H_0$$下的渐近分布**: 当$$n \rightarrow \infty$$时，$$\chi^2 \rightarrow \chi^2(rs-1)$$。
* **拒绝域**:
    $$D=\{\chi^{2}>\chi_{\alpha}^{2}(rs-1)\}$$

#### 2. 边际概率 $$p_{i\cdot}, p_{\cdot j}$$ 未知时
用极大似然估计 (MLE) 估计未知边际概率：
$$\hat{p}_{i\cdot}=\frac{n_{i\cdot}}{n}, \quad \hat{p}_{\cdot j}=\frac{n_{\cdot j}}{n}$$
* **检验统计量**:
    $$\chi^{2}=\sum_{i=1}^{r}\sum_{j=1}^{s}\frac{(n_{ij}-n\hat{p}_{i\cdot}\hat{p}_{\cdot j})^{2}}{n\hat{p}_{i\cdot}\hat{p}_{\cdot j}}$$
* **$$H_0$$下的渐近分布**: 其极限分布为$$\chi^2$$分布，自由度为 $$rs-(r+s-2)-1 = (r-1)(s-1)$$。
* **拒绝域**:
    $$D=\{\chi^{2}>\chi_{\alpha}^{2}((r-1)(s-1))\}$$
